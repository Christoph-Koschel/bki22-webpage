cd <dir|..> change directory
Changes the current directory
'..' >> Geht ein Ordner zurück
cat <file> Zeigt den Inhalt einer Datei. (Funktioniert nur mit ASCII Dateien)
touch <file> Erzeug einen neue blank datei falls diese nicht existiert
rm <item> Entfernt ein fs item
Ordner müssen leer sein
rmdir <folder> Entfernt den Ordner <folder> falls dieser leer ist
ls <options> Listet die fs items im aktuellen verzeichnis auf
ssh <host> versucht eine ssh verbindung mit dem <host> aufzubauen
ping <host> Pingt eine Server an und gibt die Zeit zurück
trace <host> Pingt einen Server an und gibt die Zeiten zurück.
Zusätzlich werden alle Router & zwischen Verbindungen gezeigt
exit Beendet das Terminal
^C Beendet den aktuellen Prozess und kehrt zum eingabe Prozess zurück (Beendet nicht die Konsole)
