In dieser Sektion erstellen wie ein kleines Backup system mit eine Simplen CLI-GUI was nur aus "echo"s und "read"s besteht

Grundbefehle

Befehl Beschreibung
echo <msg> Gibt einen String an das Konsolen fenster aus
read [-p <prefix>] <var> Liest die eingabe von einem benutzer und speichert sie in die var wenn man vor der lesung noch was ausgeben möchten muss man einen Prefix setzen
tar [...options] Wird zu zippen von dateien benutz (später kommen noch die optionen dazu)

Menu bauen

In dieser Sektion bauen wir das menu das einfache Hauptmenü
Dafür geben wir einfach eine Simple reihe an "echo"s aus die allein eine ähnliche struktur habe
Um das Shell-Script zu vereinfachen wrappen wir das gesamte Menu in eine Funktion die wir Menu nennen 


# ...
function menu() {
	echo "|------------------------|"
	echo "|  Hauptmenu             |"
	echo "|------------------------|"
	echo "|  (1) Backup erstellen  |"
	echo "|  (2) Backup restoren   |"
	echo "|  (3) Backup listen     |"
	read -p "|  Eingabe > " $CMD
}				
# ...

Damit können wir nun jedes mal wenn wir die Funktion menu aufrufen. Die das Menu baut und
die eingabe des User erwartet
