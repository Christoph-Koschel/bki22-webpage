Java, was ist das

Zu einem ist Java eine Object orientierte Programmiersprache (java-lang), jedoch gibt es auch eine Java-Virtual-Machine (kurz: JVM)
Was ist da jetzt aber der Unterschied?
Ist Java interpretiert oder kompiliert?

Kurz gesagt gibt es Java in zwei Formen:
Java Lang
Java als Programmiersprache ist eine kompilierte Sprache die Java Bytecode (.class files) generiert.
Java Bytecode
Der generierte Bytecode wird dann von der JVM interpretiert. Dabei muss der Bytecode nicht umbedingt von der Sprache Java kommen.
Es gibt genau so andere Sprachen die Java Bytecode produzieren (z.B. Kotlin).

Sie sehen man muss genau unterscheiden zwischen Java Lang und JVM.

Das OOP Konzept

Java hat das Object Orientierte Programmierparadigma
Aber was heißt das eigentlich?

Beim OOP unterscheidet man hauptsächlich zwischen Klassen und deren Instanzen.
Stellen sie sich ein leeres Blatt vor. Sie zeichen auf dieses Blatt einen Würfel. Um diesen aber sauber zu zeichnen brauchen sie bestimmte Werte.
Beachten sie dabei auch die verschiedene Eigenschaften des Würfels (Ecken, Kanten, etc.).
Zu letzt bauen sie sich den Würfel, den sie auf das Blatt gezeichnet haben.

Was sie hoffentlich gerade im Kopf durchgespielt haben, ist dass fundamentale Prinzip des OOPs.
Dabei müssen sie beachten das man ihren Würfel beliebig oft bauen kann. Zumindest so lang bis ihnen die Ressourcen ausgehen.
Kurz gesagt sie haben ein Bauplan (Klasse) geschaffen und konnten mit diesem mehrmals den gleichen Würfel bauen (Instanzen)

Natürlich bestehen Klassen aus viel mehr jedoch ist das Prinzip fast immer das gleiche.
