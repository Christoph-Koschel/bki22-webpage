Command Description
switch> enable Wechselt in denn Administrator Modus
switch# disable Wechselt in denn Benutzer Modus
switch# configure terminal Wechselt in denn Konfigurations Modus mit dem Terminal Interface
switch# hostname >name> Setzt den Hostname
switch(config)# enable secret <psw> Setzt <psw> als Administrator password
switch(config)# line console <index> Wechselt die Konfiguaration Optionen für den Consolen Input mit der Nummer <index>
switch(config-line)# password <psw> Initzalisiert ein Passwort <psw>
switch(config-line)# login Setzt das Initalisierte Passwort als Login Passwort
switch(config)# end Beendet den Konfigurations Modus und Wechselt in den Administrator Modus zurück
<Modus*>exit Wechselt denn Modus zum letzen Modus zurück (Beendet den Prozess)
switch(config)#line vty 0 4 Selektiert denn Port 0 bis 4 zu einer Gruppe
switch(config)#no ip domain lookup Versucht nicht mehr bei einer Fehleingabe eine Namensauflösung durchzuführen
switch(line)#password >password> Setzt das Passwort vom Interface
switch(config)#interface vlan 1 Wählt das erste vlan interface aus un wechselt in den Konfigurations Modus
switch(config-if)#ip address <ip> <submask> Setzt die Ip & Submask von dem Ausgewählten Interface
switch(config-if)# no shutdown Verhindert dass das interface herunterfährt
switch(config)#enable password encryption Aktiviert die Passwort verschlüssenlung
switch#show running-config Zeigt alle Konfiguarationen des Switches
